The lightweight HTTP toolkit for Kotlin applications

http4k provides a functional and consistent way to serve and consume HTTP services.

Understand our
main features

Small
footprint

The core library has no other dependencies and it's enough to get started.

Highly
extendable

Template engines, rendering formats, operability and resilience features testing tools... http4k provides 30+ integrations for developers to choose from.

Testability as a
design principle

Testing can be done in-memory without actual running servers or network communication. Various approaches such as Approval and Chaos testing are also supported.

community
